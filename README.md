# SolidToFluid

## Project introduction
Hello! This is my project of physically-based animation.

## Operation
Mouse movement: Camera direction 
Mouse scroll: Zoom in / out
W: Moving forward
A: Moving backward
S: Moving to the left
D: Moving to the right
Q: Move up
E: Move down
